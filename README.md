# ToDoList App

## Libraries Used

- **React**
- **Zustand**
- **Ant Design**
- **Axios**
- **React Infinite Scroll Component**
- **React Testing Library**
- **Vitest**

## Installation and Setup

1. **Clone the repository**
   ```sh
   git clone https://github.com/constvict/Todo-List.git
   cd Todo-List
   ```
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the application**
   ```sh
   npm run dev
   ```
4. **Run tests**
   ```sh
   npx vitest
   ```
